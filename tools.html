Torus Tools

We are working on a open source tool kit to help you have a better developer experience as a front end developer when working with static websites.

CLI with Dev Server

An intuitive and easy to use full feature command line interface for Torus Tools made with OCLIF. Provides a dev server with live reload. Just run `Torus Start`.

Form Generator

Super Easy Forms is a node module that generates responsive HTML forms with serverless backends in an instant for free.

www.supereasyforms.com

Localization

A node module that automatically localizes and translates static sites. It may also work as a very basic content management system.

www.github.com/torus-tools/localize

Audit

Audit your site pages with Lighthouse 6 in development mode and save the results locally.

www.github.com/torus-tools/seo_audit

Optimization

Use the power of webpack to compress/minify your static assets with a single command.

www.github.com/torus-tools/seo_fix
