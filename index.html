How it Works

Torus Hosting is made up of a full feature command line interface and 3 SDK's; Stack, Domains, and Content. The CLI facilitates the interaction with the SDKs while the SDKs standardizes the different services among the different cloud providers and domain registrars.

1. Stack:

A promise-based javascript SDK that generates and deploys JSON cloudformation templates for static websites in AWS. It uses the AWS SDK to create, and execute changesets in a particular sequence that enables automation of the entire process while maintaining a short deployment time.

2. Domains:

A promise-based javascript SDK for managing nameserver and DNS operations accorss multiple domain name registrars.

3. Content:

A promise-based javascript SDK for simultaneously updating content in s3 and cache in cloudfront.
